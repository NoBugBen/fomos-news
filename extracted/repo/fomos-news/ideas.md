# Fomos News — 设计理念探索

## 方案一：黑客帝国终端美学 (Matrix Terminal Aesthetics)

<response>
<text>
**Design Movement**: 赛博朋克终端主义 (Cyberpunk Terminalism)

**Core Principles**:
1. 所有信息以"终端输出"形式呈现，带有打字机动效和光标闪烁
2. 数据可视化采用ASCII字符艺术与霓虹光晕叠加
3. 布局打破传统网格，采用不对称的"数据流"结构
4. 每个交互都有"系统响应"的反馈感

**Color Philosophy**:
- 主色：#00FF41（黑客帝国绿）
- 背景：#0D0D0D（极深黑）
- 次色：#003B00（深绿阴影）
- 强调：#FF0080（霓虹粉，用于警告/多空指数）
- 文字：#E0FFE0（淡绿白）

**Layout Paradigm**:
- 左侧固定"系统终端"导航栏（模拟命令行菜单）
- 主内容区采用"数据流瀑布"布局，新闻卡片像终端输出行一样排列
- 顶部横幅显示实时"系统状态"（恐慌指数、多空概率）

**Signature Elements**:
1. 扫描线叠加效果（CSS scanlines）
2. 文字发光效果（text-shadow neon glow）
3. 边框采用虚线"数据包"动画

**Interaction Philosophy**:
- 悬停触发"系统扫描"动画
- 点击产生"矩阵雨"粒子效果
- 页面切换模拟"终端清屏"

**Animation**:
- 打字机文字逐字显示
- 矩阵雨背景（canvas动画）
- 数据加载时的进度条扫描动效

**Typography System**:
- 标题：JetBrains Mono（等宽，黑客感）
- 正文：Space Mono（等宽，终端感）
- 数据：Courier New（经典终端字体）
</text>

## 方案二：赛博哲学极简主义 (Cyber-Philosophy Minimalism)

<response>
<text>
**Design Movement**: 数字禅宗 (Digital Zen)

**Core Principles**:
1. 极简布局中蕴含深度信息密度
2. 霓虹色彩作为"信号"而非装饰
3. 大量负空间创造"数字虚空"感
4. 信息层级通过发光强度而非尺寸区分

**Color Philosophy**:
- 主色：#39FF14（霓虹绿）
- 背景：#050505（近黑）
- 次色：#1A1A2E（深蓝黑）
- 强调：#00D4FF（赛博蓝）
- 危险：#FF3333（警报红）

**Layout Paradigm**:
- 全宽不对称布局，左1/3为导航+指标，右2/3为内容流
- 内容卡片无边框，靠光晕和间距区分
- 滚动触发渐进式内容揭示

**Signature Elements**:
1. 细线分隔符带流动光点动画
2. 数字指标以大字号"浮雕"显示
3. 卡片悬停时边缘发光

**Interaction Philosophy**:
- 最小化交互，每次操作都有意义
- 滚动即是探索，无需额外点击

**Animation**:
- 元素入场：从左侧滑入，带轻微模糊
- 数字变化：滚动计数动画
- 背景：极慢速移动的粒子网格

**Typography System**:
- 标题：Orbitron（科幻感强）
- 正文：IBM Plex Mono（可读性好的等宽字体）
- 数据：Share Tech Mono
</text>

## 方案三：神经网络数据流 (Neural Network Data Stream) ✅ 选定

<response>
<text>
**Design Movement**: 神经赛博朋克 (Neural Cyberpunk)

**Core Principles**:
1. 界面模拟"神经网络节点"的信息流动
2. 暗色背景上的霓虹绿作为"激活信号"
3. 不对称的"数据面板"布局，模拟多屏监控系统
4. 明暗模式切换模拟"白天/夜间运行模式"

**Color Philosophy**:
- 暗模式主色：#00FF88（明亮神经绿）
- 暗模式背景：#080C0A（深黑绿底）
- 亮模式主色：#006633（深绿）
- 亮模式背景：#F0FFF4（极淡绿白）
- 多头（牛）：#00FF88 / #22C55E
- 空头（熊）：#FF3366 / #EF4444
- 数据蓝：#00D4FF
- 警告橙：#FF8C00

**Layout Paradigm**:
- 顶部固定导航栏（系统状态栏风格）
- 主页采用"监控面板"布局：左侧新闻流 + 右侧实时指标
- 简报页采用全宽"终端输出"风格
- 排行榜采用"竞技场"风格的表格

**Signature Elements**:
1. 细微的扫描线背景纹理
2. 卡片边框带流动光点（CSS动画）
3. 数字数据以等宽字体+发光效果显示

**Interaction Philosophy**:
- 悬停时卡片边框激活（绿色发光）
- 点击有"信号传输"的涟漪效果
- 主题切换有"系统重启"的过渡动画

**Animation**:
- 页面加载：顶部进度条扫描
- 卡片入场：交错淡入+轻微上移
- 数字指标：计数滚动动画
- 背景：极慢速的网格线移动

**Typography System**:
- 标题：JetBrains Mono Bold（等宽，黑客感）
- 正文：Inter（高可读性，现代感）
- 数据/代码：JetBrains Mono Regular
- 中文：系统默认中文字体（保证可读性）
</text>
